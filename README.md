# Exploratory Data Analysis (EDA) on Multiple Datasets

## Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on multiple datasets to understand their structure, statistical properties, and data distributions.
The analysis is implemented using **Python**, **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn** within a Jupyter Notebook environment.

The datasets used in this project represent **classic machine learning data** and **real-world entertainment industry data**, making the analysis relevant for both **academic learning and practical ML applications**.

##  Notebook Description

###  Task3_Exploratory_Data_Analysis.ipynb

This notebook performs a complete **EDA workflow**, including:

* Loading and inspecting datasets
* Understanding dataset dimensions and data types

##  Dataset Descriptions

###  1. Iris Dataset

**File:** `IRIS.csv`
**Type:** Classification Dataset

#### Features

| Column      | Description                                  |
| ----------- | -------------------------------------------- |
| SepalLength | Sepal length in cm                           |
| SepalWidth  | Sepal width in cm                            |
| PetalLength | Petal length in cm                           |
| PetalWidth  | Petal width in cm                            |
| Species     | Flower class (Setosa, Versicolor, Virginica) |

#### Use Cases

* Classification algorithms
* Data visualization practice
* Feature analysis
* Introductory machine learning experiments



###  2. Netflix Titles Dataset

**File:** `netflix_titles.csv`
**Type:** Real-world Entertainment Dataset

#### Key Attributes

| Column       | Description                   |
| ------------ | ----------------------------- |
| show_id      | Unique show identifier        |
| type         | Movie or TV Show              |
| title        | Title name                    |
| director     | Director of the content       |
| cast         | Main cast                     |
| country      | Country of production         |
| date_added   | Date added to Netflix         |
| release_year | Year of release               |
| rating       | Content rating                |
| duration     | Duration or number of seasons |
| listed_in    | Genre/category                |
| description  | Short description             |

#### Use Cases

* Real-world EDA practice
* Content trend analysis
* Business analytics
* Recommendation system preprocessing
